SKILLS



HTML

HTML which is also called as Hypertext Markup Language is a basic and standard language used to create a webpage. This is done with the help of many tags and attributes. HTML is used to create a basic structure of a webpage.These tags define elements such as headings, paragraphs, images, links, and more. HTML is essential for building the basic structure of a website.

CSS

CSS which is also known as Cascading Style Sheet is a computer language for laying out and structuring web pages. It is use to make the HTML document or the structure of a webpage to be more colorful and be dynamic. It is used for styling of the HTML document.
